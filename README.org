#+TITLE: Tiny_Org_Mode ein webxdc Org Editor

* Beschreibung
Ein minimalistischer, lokal speichernder Org-Mode-Editor für WebXDC-Apps.

* Installation
** Direkt im Browser nutzen
- Datei ~index.html~ herunterladen und im Browser öffnen.
- Notizen werden automatisch im LocalStorage gespeichert.

** In WebXDC/Delta Chat integrieren
- Datei als ~.xdc~-Datei speichern (z. B. ~org-editor.xdc~).
- In Delta Chat oder einer WebXDC-App öffnen.

* Funktionen
** Org-Mode-Unterstützung
- Überschriften: ~* Überschrift 1~, ~** Überschrift 2~, etc.
- Aufgabenstatus: ~TODO~, ~DOING~, ~DONE~.
- Checklisten: ~- [ ] Aufgabe~, ~- [x] Erledigt~.
- Textformatierung: ~*Fett*~, ~/Kursiv/~.

** Editor-Features
- Echtzeit-Vorschau.
- Einfache Zeilenbearbeitung.
- Ein- und Ausklappen von Abschnitten.

** Export & Import
- Export als ~.org~-Datei oder in die Zwischenablage.
- Import von ~.org~- oder ~.txt~-Dateien.

** Praktische Helfer
- Zeitstempel einfügen.
- Automatische Zeilenformatierung mit Enter.

* Technische Details
- Speicherung im LocalStorage.
- Offline-fähig.
- WebXDC-kompatibel.

* Beispiel
#+BEGIN_EXAMPLE
* TODO Projekt starten
** [ ] Aufgabenliste erstellen
** [x] Design entwerfen
* Notizen
- Wichtiger Punkt: *Fett* und /kursiv/ formatieren.
#+END_EXAMPLE
